# Contributing

Thanks for your interest in improving this SDK.

## Reporting bugs

Open a [GitHub issue](https://github.com/apivault-labs/producthunt-scraper-python/issues) with:
- Python version (`python --version`)
- SDK version (`pip show producthunt-scraper`)
- The ProductHunt URL or mode that triggered the issue
- The full traceback or unexpected output

## Suggesting features

- **Client-side improvements** — open an issue here
- **New extracted fields or modes** — those live in the underlying Apify actor; open an issue

## Development setup

```bash
git clone https://github.com/apivault-labs/producthunt-scraper-python.git
cd producthunt-scraper-python
python -m venv .venv
. .venv/bin/activate
pip install -e .
```
